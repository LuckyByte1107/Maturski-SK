Основе веб технологије

HTML • CSS • JavaScript

1. HTML - Структура веб странице

HTML (HyperText Markup Language) је основни језик за израду веб страница. Он дефинише структуру садржаја путем ознака као што су <p>, <a>, <img> и друге.

<a href="https://google.com">Линк ка Google-у</a>

2. CSS - Дизајн и изглед

CSS омогућава да стилизујемо HTML елементе. Можемо мењати боје, фонтове, величине, распореде и још много тога.

p {
  color: blue;
  font-size: 18px;
}

3. JavaScript - Интерактивност

JavaScript додаје динамику веб страници. Можемо реаговати на кликове, уносе и мењати садржај без освежавања странице.
